# 🧮 Complexly Learner — Complex Number Visualizer and Explorer

Welcome to **Complexly Learner**, a web-based interactive tool to explore and visualize complex number operations and concepts in an intuitive way.

## 🚀 Live Demo

🌐 [Click here to try it live!](https://your-render-url.onrender.com)

## 📌 Why I Built This

As a mathematics and programming enthusiast, I wanted to build something practical that helps learners **understand complex numbers visually** — not just numerically.

This project helped me:
- Strengthen my Python and Flask skills
- Work with Google Sheets API
- Understand complex mathematical concepts deeply
- Build a full-stack web app and deploy it on Render

## 📂 Features

- 🎨 Argand Plane Plotter
- 🧠 Complex Number Calculator (Add, Subtract, Multiply, Divide)
- 📏 Modulus & Argument Explorer
- 🔁 Polar ↔ Cartesian Converter
- 📐 De Moivre's Theorem Visualizer
- 🌀 Roots of Unity Visualizer
- 🧪 Complex Number Quiz (auto-evaluation + Google Sheets storage)
- 🧠 Mandelbrot Fractal Explorer (WIP)

## 🛠️ Tech Stack

- Python (Flask)
- HTML/CSS, JavaScript
- Google Sheets API
- Matplotlib, NumPy
- Hosted on **Render**
