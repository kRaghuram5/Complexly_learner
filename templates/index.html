The Complex Numbers

Mathematical Definition:

A complex number is an element of the set ℂ = { a + bi | a, b ∈ ℝ, i² = –1 }, where a is the real part, b is the imaginary part, and i is the imaginary unit satisfying i² = –1.

A complex number is a number of the form a + bi, where a and b are real numbers, and i is the imaginary unit with i² = –1. They emerged from equations like x² + 1 = 0 (where a = c = 1, b = 0) that have no solution in real numbers, since no real number squared equals –1. By introducing i, mathematicians created a number system that solves the unsolvable.

Key Properties

  • Closure: The sum or product of two complex numbers is also a complex number.
  • Commutative Property: u + v = v + u, uv = vu.
  • Associative Property: (u + v) + w = u + (v + w), (uv)w = u(vw).
  • Distributive Property: u(v + w) = uv + uw.
  • Identities: Additive (0) and multiplicative (1) identities exist for all complex numbers.
  • Additive Inverse: For any z, there exists −z such that z + (−z) = 0.
  • Multiplicative Inverse: For z ≠ 0, z⁻¹ = (a − ib)/(a² + b²).
  • Powers of i: i² = −1, i³ = −i, i⁴ = 1 (repeats every 4 powers).
  • Modulus: |z| = √(a² + b²) — distance from origin in Argand plane.
  • Conjugate: z = a + ib ⇒ ȳ = a − ib; used in division and |z|² = zȳ.
  • Quadratic Roots: Solve negative discriminants using i.
