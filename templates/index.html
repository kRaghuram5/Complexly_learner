The Complex Numbers

Have you ever wondered why mathematicians invented a whole new kind of number?

It all starts with equations. In earlier classes, you solved linear equations (straightforward) and quadratic equations (a bit trickier). Most had real number solutions — until one day, a strange case appeared.

Consider the quadratic equation ax² + bx + c = 0. The nature of its solutions depends on the discriminant D = b² – 4ac.

  • If D > 0, there are two real roots.
  • If D = 0, there’s one real root.
  • If D < 0, there are no real roots.

Let’s take an example: x² + 1 = 0.
Here, a = 1, b = 0, c = 1, so D = 0² – 4(1)(1) = –4, which is less than zero.
Solving gives x² = –1 — a dead end in the real number system, because no real number squared gives a negative result.

So mathematicians asked: What if we create a new number whose square is –1?
They called it i, where i² = –1.

With this, numbers like a + ib became possible — a combination of a real part (a) and an imaginary part (b).
Examples: 2 + 3i, –1 + √3 i, 4 – (1/11) i.

This simple invention unlocked solutions for every quadratic equation — and gave rise to a powerful number system that appears in engineering, physics, and even digital technology.

Welcome to the world of complex numbers — born from the question: How do we solve the unsolvable?

Quick Formula Sheet


1. Standard Form:
   z = a + ib
   Real part: Re(z) = a      Imaginary part: Im(z) = b
2. Equality:
   a + ib = c + id  =>  [a = c] and [b = d]
3. Addition & Subtraction:
   z1 ± z2 = (a ± c) + i(b ± d)
4. Multiplication:
   z1 * z2 = (ac - bd) + i(ad + bc)
5. Division:
   (a + ib) / (c + id) = [(ac + bd) + i(bc - ad)] / (c² + d²)
6. Conjugate:
   z̄ = a - ib
   z * z̄ = a² + b²
7. Modulus & Argument:
   r = |z| = √(a² + b²)
   θ = arg(z) = tan⁻¹(b / a)   [adjust for quadrant]
8. Polar (Trigonometric) Form:
   z = r(cosθ + i sinθ)
   Euler's form: z = r e^(iθ)
9. Conversion:
   Rectangular → Polar: r = √(a² + b²), θ = arg(z)
   Polar → Rectangular: a = r cosθ, b = r sinθ
10. Multiplication in Polar Form:
    r1(cosθ1 + i sinθ1) × r2(cosθ2 + i sinθ2) =
    r1 r2 [cos(θ1 + θ2) + i sin(θ1 + θ2)]
11. Division in Polar Form:
    (r1 / r2) [cos(θ1 - θ2) + i sin(θ1 - θ2)]
12. Powers (De Moivre’s Theorem):
    zⁿ = rⁿ [cos(nθ) + i sin(nθ)]
13. Roots of Complex Numbers:
    z_k = r^(1/n)[cos((θ + 2kπ)/n) + i sin((θ + 2kπ)/n)],k=0,1,...,n-1
14. Inverse (Reciprocal):
    z⁻¹ = z̄ / |z|² = a/(a²+b²) - i b/(a²+b²)
15. Special Powers of i:
    i⁰ = 1, i¹ = i, i² = -1, i³ = -i, i⁴ = 1 (repeat every 4)

Key Properties

  • Closure: The sum or product of two complex numbers is also a complex number.
  • Commutative Property: u + v = v + u, uv = vu.
  • Associative Property: (u + v) + w = u + (v + w), (uv)w = u(vw).
  • Distributive Property: u(v + w) = uv + uw.
  • Identities: Additive (0) and multiplicative (1) identities exist for all complex numbers.
  • Additive Inverse: For any z, there exists −z such that z + (−z) = 0.
  • Multiplicative Inverse: For z ≠ 0, z⁻¹ = (a − ib)/(a² + b²).
  • Powers of i: i² = −1, i³ = −i, i⁴ = 1 (repeats every 4 powers).
  • Modulus: |z| = √(a² + b²) — distance from origin in Argand plane.
  • Conjugate: z = a + ib ⇒ ȳ = a − ib; used in division and |z|² = zȳ.
  • Quadratic Roots: Solve negative discriminants using i.

Uses of Complex Numbers

  • Engineering & Electronics: AC circuits, control systems, signal processing.
  • Physics: Quantum mechanics, wave theory, fluid dynamics.
  • Computer Science: Image processing, simulations, graphics.
  • Music & Audio: Waveform analysis, sound synthesis.
